# About d-money-user project
D-money-user is a system where user can be created, updated, deleted and searched by different criteria.

# Tools used:
- postman
- newman
- nodejs
- git

# How to run?
- Clone this project
- PUT .env file (ask the repo owner)
- Give following command:
- ``` npm i ```
- ``` npm test ```
